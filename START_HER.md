## 💡 Pro tips:

1. **Test lokalt først**
   ```bash
   pytest tests/ -v
   ```

2. **Se test coverage**
   ```bash
   pytest tests/ --cov=my_calculator
   ```

3. **Check kode før push**
   ```bash
   flake8 my_calculator/
   ```
